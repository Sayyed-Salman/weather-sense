WeatherSense

Weather forecast based on your location

Try Now

This Flask application utilizes Python, Flask, and Tailwind CSS to create a user-friendly interface for displaying the current weather based on the user's location. The weather data is obtained from the Open-Meteo weather API.

When a user accesses the application, the Flask server retrieves the user's location information, such as latitude and longitude, using browser geolocation. Then, it makes a request to the Open-Meteo weather API to fetch the current weather data for that location.

The weather information, including temperature, humidity, wind speed, and conditions, is then displayed on the webpage in a visually appealing manner using Tailwind CSS for styling.

The Flask application is hosted on PythonAnywhere, a platform that allows easy deployment and hosting of Flask applications. Users can access the application through its PythonAnywhere URL to view the current weather based on their location.

This application demonstrates the power of Flask and Python in combination with Tailwind CSS to create dynamic and responsive web applications, providing users with up-to-date weather information at their fingertips.
